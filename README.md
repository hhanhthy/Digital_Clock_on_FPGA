# Digital_Clock_on_FPGA
youtube: https://www.youtube.com/watch?v=I9IBpSwVAUA

## Overview
* **This project implements a Digital Clock with Alarm and Snooze functions on FPGA using Verilog HDL. The system provides a hardware-based user interface through switches and push-buttons, allowing users to view real-time time information, configure clock settings, set alarms, and manage snooze operations. Unlike software-based clock implementations, this design emphasizes synchronous digital system design, finite state machine (FSM) control, and real-time hardware processing on an FPGA platform. The project demonstrates the integration of multiple digital design concepts including clock division, debouncing, multiplexed display control, finite state machine architecture, and alarm management.
